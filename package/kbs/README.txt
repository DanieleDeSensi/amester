amester-kbs.tcl is based on
https://sourceforge.net/projects/kbskit/files/kbs/0.4.8/kbs.tcl

It builds the kbsmk8.6-bi binary that is the Tcl runtime used for Amester.
"bi" means batteries included and indicates that the Tcl binary includes popular Tcl extensions.  
Amester uses many of these extensions, including tls, rbc, etc.

The build script is known to work on the following systems:
  Linux x86_64
  Linux POWER7
